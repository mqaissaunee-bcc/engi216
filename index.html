Course Home

Welcome to the interactive course

This course presents the kinematics, dynamics, and synthesis of planar mechanisms and machines as a series of self-contained, interactive lecture modules. Each chapter opens in your browser, runs offline, and includes live mechanism simulations, worked examples, self-checks, embedded video, a glossary, and a printable formula reference.

Approach

Synthesis is taught first, then analysis — so the mathematics behind mechanism design makes sense before you analyze existing machines.

Core method

The Vector Loop Method and kinematic coefficients run through the whole sequence, giving a single consistent, programmable framework.

How to use

Work through chapters in order. Each module is one browser tab; the simulators and self-checks are designed for active, hands-on practice.

What you'll be able to do

Course learning objectives

By the end of this course, students will be able to:

  • Identify links and joints and draw skeleton (kinematic) diagrams of real machines.
  • Compute mobility with Gruebler's criterion and classify four-bars with Grashof's criterion.
  • Set up and solve vector loop equations for the position of planar mechanisms.
  • Handle rolling-contact constraints, including gears and friction wheels.
  • Use first- and second-order kinematic coefficients for velocity and acceleration.
  • Solve the inverse dynamics problem to find the driving torque for a prescribed motion.
  • Account for friction in pin, pin-in-slot, and sliding joints.
  • Apply the power equation and evaluate mechanical advantage and efficiency.
  • Synthesize four-bar and crank-slider mechanisms with Freudenstein's equation.
  • Design mechanisms for rigid-body guidance through multiple prescribed positions.
About these modules

How the modules are built

Each chapter is a single self-contained HTML file: all styling and interactivity are inline, so the modules run in any modern browser, work offline, and embed cleanly in a learning-management system. They follow a consistent engineering-blueprint design and are built to meet WCAG 2.1 AA accessibility guidance.

Every chapter module includes: a guided lecture, interactive mechanism simulators, worked examples, self-check questions, an embedded video page, a searchable glossary, categorized external resources, a printable quick-reference formula card, an objectives recap, and an attribution block.
Accessibility & new tabs

Chapter links open in a new tab and are marked with a visible "↗ opens in a new tab" indicator and a screen-reader label so the behavior is never a surprise. All modules support keyboard navigation, dark mode, and reduced-motion preferences.
